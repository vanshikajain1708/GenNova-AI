# GenNova AI 🚀

A decentralized, multi-agent workspace designed for intelligent workflows and real-time analytics. Built as a solo project for the Gen AI Academy APAC Program.

## 🌟 Key Features

* **Multi-Agent Orchestration** – Specialized agents for diagnosis, planning, analytics, and recommendations.
* **Secure Authentication** – User login and profile management system.
* **Multi-Modal Ingestion** – Supports Voice, PDF, and Image inputs.
* **Real-Time Analytics** – Interactive dashboard with executive PDF report generation.
* **Modern UI/UX** – Responsive and intuitive interface built with React.
* **AI-Powered Insights** – Actionable recommendations generated through intelligent agent collaboration.

## 🛠 Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Build Tool:** Vite
* **Backend:** Node.js
* **Version Control:** Git & GitHub

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/vanshikajain1708/GenNova-AI.git
cd GenNova-AI
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
